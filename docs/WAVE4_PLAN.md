# Wave 4 — Task Blo: Полноценный продукт

## Цель Wave 4

Превратить Task Blo из «бота для задач» в полноценный продукт с завершённым UX, продающей презентацией и Premium-воронкой. После Wave 4 пользователь должен видеть Task Blo как готовый, отполированный инструмент — не бета-версию.

---

## 1. Презентация и онбординг бота

### 1.1 /start — полная переделка

| Элемент | Текущее состояние | Wave 4 |
|---------|------------------|--------|
| Картинка | Одна статичная | 3-4 слайда-карусель (Telegram album) с ключевыми фичами |
| Описание | Длинный текст | Краткие bullet points (≤ 3 строки на слайд) |
| CTA-кнопка | «Open app» | «Открыть Task Blo» + «Попробовать Premium бесплатно» |

**Карусель слайдов:**
1. **Задачи** — «Записывай задачи голосом и текстом. Бот поймёт.» + скриншот бота
2. **Питомец** — «Твой виртуальный питомец растёт вместе с тобой. Выполняй задачи — он будет счастлив!»
3. **Геймификация** — «Ежедневные квесты, колесо удачи, достижения и монеты.»
4. **Premium** — «Безлимит задач, голосовой ввод, расширенные напоминания — от 99 ⭐/мес.»

### 1.2 Bot Description & About

- **Bot Description** (Telegram BotFather): краткое описание ≤ 512 символов, все ключевые фичи
- **About** (BotFather → /setabouttext): 1-2 предложения + ссылка на Mini App
- **Bot Profile Photo**: обновлённое лого Task Blo (питомец с задачами)

### 1.3 Bot Menu Commands

Обновить `/setcommands` для быстрого доступа:

```
start - Запустить бота
app - Открыть Mini App
premium - Узнать о Premium
quests - Ежедневные квесты
pet - Мой питомец
help - Помощь
```

## 2. Premium-воронка

### 2.1 Soft-баннеры (ненавязчивые)

Показывать мягкие напоминания о Premium в контексте использования:

- **При достижении лимита задач (10/день)**: «Хочешь больше? С Premium лимита нет»
- **При попытке голосового ввода**: уже реализовано в Wave 3
- **После 7-го дня стрика**: «Ого, 7 дней подряд! С Premium ты получишь x2 монет за стрик»
- **В колесе удачи**: «Premium открывает легендарные призы»

### 2.2 Paywall-экран в Mini App

Добавить красивый экран `/premium` в Mini App:

- Карточка сравнения Free vs Premium
- Анимированные иконки для каждой фичи
- Кнопка «Попробовать 3 дня бесплатно» (Trial)
- Отзывы / социальное доказательство (можно фейковые на старте)

### 2.3 Push-уведомления (эмоциональные)

Бот отправляет сообщения в Telegram:

| Триггер | Сообщение (RU) |
|---------|---------------|
| Утро (9:00) | 🌅 «У тебя 3 задачи на сегодня. Твой питомец ждёт!» |
| Не заходил 2 дня | 😢 «{pet_name} скучает… Загляни, у тебя ждут квесты!» |
| Стрик на грани (23:00) | 🔥 «Не потеряй свой стрик {N} дней! Осталась 1 задача» |
| Достижение разблокировано | 🏆 «Разблокировано: {achievement_name}! Загляни посмотреть» |
| Новый уровень питомца | 🎉 «{pet_name} достиг стадии {stage}! Посмотри!» |

**Частота**: максимум 2 push-уведомления в день, с настройкой отключения.

## 3. Улучшения бот-меню (вне Mini App)

### 3.1 Quick-Task через Inline кнопки

После создания задачи показывать Reply Keyboard:

```
[ Ещё задачу ]  [ Открыть приложение ]
```

### 3.2 Reply Keyboard для быстрого ввода

При входе в бот (после /start) показывать стандартную клавиатуру:

```
[ 📝 Новая задача ]  [ 📋 Мои задачи ]
[ 🐾 Питомец ]       [ ⚙ Настройки ]
```

### 3.3 Inline-режим (@taskblo_bot)

Поддержка инлайн-режима: пользователь пишет `@taskblo_bot купить молоко` в любом чате → бот предлагает создать задачу.

## 4. Улучшения Mini App

### 4.1 Онбординг-тур (первый вход)

3 шага с анимацией:
1. «Привет! Я помогу тебе с задачами» → тап
2. «У тебя есть питомец — он растёт, когда ты выполняешь задачи» → тап
3. «Готов? Начнём!» → переход на главную

### 4.2 Настройки пользователя

Страница `/settings`:
- Язык (RU / EN)
- Часовой пояс
- Уведомления (вкл/выкл утренние, вечерние, стрик-напоминания)
- Тема (auto / light / dark)
- Сброс стрика (Premium)

### 4.3 Статистика

Страница `/stats`:
- Задачи за неделю/месяц (график)
- Лучший стрик
- Категории (pie chart)
- Монеты заработано всего

## 5. Техническая часть

### 5.1 Scheduler (фоновые задачи)

Для push-уведомлений нужен APScheduler или Celery:
- Утренние / вечерние уведомления по часовому поясу пользователя
- Проверка стрика в 23:00 по TZ пользователя
- Очистка истёкших данных

### 5.2 Analytics

- Интеграция Telegram Analytics (BOT_TOKEN metrics)
- Логирование ключевых событий: task_created, quest_completed, premium_purchased
- Funnel: start → first_task → 7_day_retention → premium

### 5.3 Rate Limiting

- Антиспам для API (уже частично есть)
- Ограничение на push-уведомления (max 2/день)

## Резюме

Wave 4 фокусируется на трёх китах:
1. **Презентация** — пользователь с первого взгляда понимает, что Task Blo — полноценный продукт
2. **Монетизация** — мягкая воронка к Premium через контекстные soft-баннеры и push
